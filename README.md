# TFE-Tarefa2

Projeto desenvolvido utilizando React, TypeScript, Vite e Bootstrap 5 para a disciplina Tecnologias Front-End.

## Funcionalidades
- Exibição de editais de monitoria
- Renderização condicional de status
- Componentização React
- Props tipadas com TypeScript
- Layout responsivo com Bootstrap

## Executar

```bash
npm install
npm run dev
```
